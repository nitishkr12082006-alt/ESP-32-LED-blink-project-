# ESP-32-LED-blink-project-
# ESP32 LED Blink

## Components
- ESP32 Dev Board
- LED
- 220Ω Resistor
- Breadboard
- Jumper Wires

## Connections
LED Anode -> GPIO2
LED Cathode -> GND (through 220Ω resistor)

## Working
The ESP32 turns the LED ON for 1 second and OFF for 1 second continuously.

## Software
- Arduino IDE
- Wokwi Simulator
